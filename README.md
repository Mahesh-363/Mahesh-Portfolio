## ⚙️ Tech Stack

### 🧠 Backend & Systems

* Python (Django, Django REST Framework)
* WebSockets (Django Channels)
* Redis (Pub/Sub, caching)
* Celery (background jobs)
* PostgreSQL

### ☁️ Cloud & DevOps

* AWS (EC2, S3, IAM, VPC)
* Docker (containerization)
* Nginx (reverse proxy)

### 🎨 Frontend

* HTML5, CSS3 (Custom design system)
* JavaScript (Vanilla)
* Responsive UI / Animations

## ✨ Key Features

* ⚡ Fully custom UI (no templates used)
* 🎨 Design system using CSS variables (tokens)
* 🚀 Smooth animations & scroll-based reveals
* 💡 Interactive UI (hover effects, glow system)
* 📱 Fully responsive across devices
* 🔥 Premium hero section with canvas effects
* 🧠 Structured sections:

  * About
  * Experience
  * Projects
  * Skills (Bento UI)
  * Contact
